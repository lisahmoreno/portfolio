What is a Claude Code skill?

Claude Code is a terminal-based AI coding assistant. Skills are custom slash commands defined in Markdown files that give Claude a specific workflow to follow — like a playbook you can invoke on demand. When you type /research-spec, Claude reads the skill file and runs its multi-phase process automatically.

This is interesting from a PM perspective because it's essentially encoding process design into a tool — the same way you'd document a process for a team, except the "team member" is an AI that will follow it consistently.

The two skills

/research-spec

A multi-phase workflow for technical investigations that enforces a self-review gate before publishing anything externally (Confluence, Slack, etc.). Every claim needs a source citation. Every section gets a confidence score. The AI cannot publish until it has reviewed its own output against a checklist.

/stakeholder-update

A writing assistant for stakeholder communications. Applies the Pyramid Principle — lead with the key message, support with 2–5 points, add detail only where necessary. Enforces a direct, non-defensive tone and explicitly forbids common AI filler phrases.
